# timeattck 프로젝트

## 개요
- Swift/SwiftUI iOS 앱 (iPhone 13 Mini)
- 개인 시간 추적 앱

## 파일 구조
- `timeattckApp.swift` - 앱 진입점
- `ContentView.swift` - 메인 탭 뷰
- `TimerView.swift` - 타이머 기능
- `ProjectView.swift` - 프로젝트 관리
- `ReportView.swift` - 리포트 뷰
- `DataModel.swift` - 데이터 모델

## 규칙
- SwiftUI만 사용
- 코드 수정 시 전체 파일로 교체해서 보여줄 것
- 설명은 한국어로
